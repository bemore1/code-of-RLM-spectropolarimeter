This repository contains the MATLAB implementation for the numerical methods presented in our Nature Communications paper: "Resonant microtaper leaky-mode computational spectropolarimetry with tens of femtometers spectral resolution and full Stokes measurement".
The core of this toolkit is a solver for nonlinear systems of equations leveraging Compressed Sensing (CS) principles via the CVX optimization framework. It is designed to handle underdetermined nonlinear problems in computational system.


Prerequisites
MATLAB (Tested on R2022b)
CVX  Bundle (Installed and configured in your MATLAB path)
